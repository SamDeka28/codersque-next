# Codersque-Site

## Prerequisites

- **Node.js**: >= 22.0.0 (LTS)
- **npm**: >= 10.0.0

## Getting Started

### Using NVM (Node Version Manager)

If you're using NVM, the project includes a `.nvmrc` file. Simply run:

```bash
nvm use
```

This will automatically switch to Node.js 22 LTS.

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Production

```bash
npm start
```
